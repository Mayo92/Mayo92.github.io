常用命令

  • git init：创建git仓库
  • git status：查看仓库状态
  • git diff+文件名：查看修改了哪些内容
  • git add+文件名：放入到暂存区
  • git commit：提交到代码分支（默认master）
  • git clone+地址：克隆远程代码
  • git branch：查看当前分支
  • git checkout：切换分支
