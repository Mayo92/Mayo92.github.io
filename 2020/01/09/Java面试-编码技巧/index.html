边界控制-二分查找

思路

  • 规定要检查的值K可能在的数组arr内下标区间a，b
  • 计算区间a，b的中间点m
  • 若k < arr[m] ，将区间缩小为a, m，继续二分查找
  • 若k > arr[m]，将区间缩小为m, b，继续二分查找
