1. Android常用控件

1.1 Button

系统会对Button中的所有英文字母进行自动大小写转换，如果这不是你想要的效果，使用如下配置来禁用这一默认特性：android:textAllCaps=”false”

1.2 EditText

android:maxLines=”2”指定EditText的最大行数为两行，这样当输入的内容超过两行时，文本就会向上滚动，而EditText则不会再继续拉伸。

#HEXO

1. 准备环境

安装nodejs，查看node版本：
node -v
安装git，查看git版本：
git --version
安装Xcode
npm安装Hexo：
sudo npm install -g hexo(官网有坑)
初始化，在指定目录创建所需要的文件：
  • hexo init <文件夹名称>
  • cd <文件夹名称>
  • npm install
文件目录含义
  • _config.yml 配置信息
  • package.json 应用程序信息
  • scaffolds 模板文件夹
  • source 资源文件夹. _posts 目录下为你已经发表的文章. _drafts 目录下为你存放的草稿文章.
  • themes 主题文件夹
