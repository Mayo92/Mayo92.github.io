Java简介

一门面向对象的程序设计语言

JVM

Java Virtual Machine，Java虚拟机，JVM是Java平台无关性实现的关键。

Java程序执行过程

JDK

Java Development Kit

Java语言的软件开发工具包

两个主要组件

  • javac–编译器，将源程序转成字节码
  • java–运行编译后的java程序（.class后缀的）

JRE

  • Java Runtime Environment
  • 包括Java虚拟机(JVM)、Java核心类库和支持文件
  • 如果只需要运行Java程序，下载并安装JRE即可
  • 如果要开发Java软件，需要下载JDK
  • JDK中附带JRE
