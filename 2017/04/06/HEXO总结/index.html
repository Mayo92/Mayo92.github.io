npm安装Hexo：
初始化，在指定目录创建所需要的文件：
  • hexo init <文件夹名称>
  • cd <文件夹名称>
  • npm install
文件目录含义
  • _config.yml 配置信息
  • package.json 应用程序信息
  • scaffolds 模板文件夹
  • source 资源文件夹. _posts 目录下为你已经发表的文章. _drafts 目录下为你存放的草稿文章.
  • themes 主题文件夹
