激活过程

1.访达，服务，服务偏好

2.文件和文件夹，新建位于文件夹位置的终端窗口

3.在桌面，右键激活的文件，新建位于文件夹位置的终端窗口

4.chmod +x mac.sh

5../mac.sh

6.重启软件

7.进入Activation_code文件夹

8.打开要激活的软件对应的文本复制，粘贴到软件ActivationCode处，点击Activate

key is invalid

1.进入micool_macconfig，configfile，jetbra，找到ja-netfilter.jar

2.拖入终端，复制路径

3.先用临时账户激活 1148696639@qq.com密码99936869wu

4.打开软件help，edit custom vm options

5.输入
–add-opens=java.base/jdk.internal.org.objectweb.asm=ALL-UNNAMED
–add-opens=java.base/jdk.internal.org.objectweb.asm.tree=ALL-UNNAMED
再加一行上面jar包路径，-javaagent:填ja-netfilter.jar的路径=jetbrains

6.重启软件，输入软件对应激活码
