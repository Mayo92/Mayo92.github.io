Java环境配置

M芯片

Azul Zulu OpenJDK，下载地址 https://www.azul.com/，选择基于macOS版ARM 64位Java8

安装目录

/Library/Java/JavaVirtualMachines/zulu-8.jdk

环境变量配置

可以在终端的任意路径下使用javac和java命令

vim .zshrc

将输入法切换到英文后再按i键切换到编辑模式

export JAVA_HOME=/Library/Java/JavaVirtualMachines/zulu-8.jdk/Contents/Home
export PATH=$PATH:$JAVA_HOME/bin

按esc键退出编辑模式回到命令模式

最后输入:wq并回车保存并退出vim编辑器

source .zshrc 命令将.zshrc配置文件中的环境变量加载到shell会话中

在终端上输入 echo $JAVA_HOME命令回车后如果能够输出Java8的安装目录/Library/Java/JavaVirtualMachines/zulu-8.jdk/Contents/Home 表示JAVA_HOME环境变量配置成功

在终端上输入echo $PATH命令查看系统的PATH环境变量中如果包含了Java8的PATH环境变量则表示配置成功

DIR="$(cd "$(dirname "$0")" && pwd)"
$DIR/build/ndk-build "$@"

DIR="$(cd "$(dirname "$0")" && pwd)"

激活过程

1.访达，服务，服务偏好

2.文件和文件夹，新建位于文件夹位置的终端窗口

3.在桌面，右键激活的文件，新建位于文件夹位置的终端窗口

4.chmod +X mac.sh

5../mac.sh

6.重启软件

7.进入Activation_code文件夹

8.打开要激活的软件对应的文本复制，粘贴到软件ActivationCode处，点击Activate

key is invalid

1.进入micool_macconfig，configfile，jetbra，找到ja-netfilter.jar

2.拖入终端，复制路径

3.先用临时账户激活 1148696639@qq.com密码99936869wu

4.打开软件help，edit custom vm options

5.输入
–add-opens=java.base/jdk.internal.org.objectweb.asm=ALL-UNNAMED
–add-opens=java.base/jdk.internal.org.objectweb.asm.tree=ALL-UNNAMED
再加一行上面jar包路径，-javaagent:填ja-netfilter.jar的路径=jetbrains

6.重启软件，输入软件对应激活码
