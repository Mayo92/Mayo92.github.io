1. Android常用控件

1.1 Button

系统会对Button中的所有英文字母进行自动大小写转换，如果这不是你想要的效果，使用如下配置来禁用这一默认特性：android:textAllCaps=”false”

1.2 EditText

android:maxLines=”2”指定EditText的最大行数为两行，这样当输入的内容超过两行时，文本就会向上滚动，而EditText则不会再继续拉伸。

1.3 ImageView

代码设置资源，imageView.setImageResource(R.drawable.img);

1.4 ProgressBar

可以给ProgressBar指定不同的样式，刚刚是圆形进度条，通过style属性可以将它指定成水平进度条，也可设置其他类型，style=”?android:attr/progressBarStyleHorizonal”,android:max=”100”

int progress = progressBar.getProgress();
progress = progress + 10;
progressBar.setProgress(progress);

2. 基本布局

2.1 线性布局LinearLayout

如果排列方向是horizontal，内部控件就不能将宽度指定为match_parent，这样的化单独一个控件就会将整个水平方向占满，其他控件就没有可放置的位置了。同理vertical，内部控件就不能将高度指定为match_parent。

android:gravity用于指定文字在控件中的对齐方式，而android:layout_gravity用于指定控件在布局中的对齐方式。需要注意，当LinearLayout的排列方向是horizontal时，只有垂直方向上的对齐方式才会生效，因为此时水平方向上的长度是不固定的，每添加一个控件水平方向上的长度都会改变，因而无法指定该方向上的对齐方式。同理，为vertical时，只有水平方向上的对齐方式才会生效。

一个weight为1，另一个width为wrap_content时，第一个会占满除了另一个的位置
