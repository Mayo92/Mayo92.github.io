public View getView(int position,View convertView,ViewGroup parent) {

    View view = LayoutInflater.from(getContext()).inflate(resorced,parent,false);
    ...
}

重写父类的一组构造方法，用于将上下文、ListView子项布局的id和数据都传递进来。getView方法在每个子项被滚动到屏幕内的时候会被调用，使用LayoutInflater来为这个子项加载我们传入的布局。LayoutInflater的inflate()方法接收三个参数，第三个参数指定false，表示只让我们在父布局中声明的layout属性生效，但不为这个View添加父布局。因为一旦有父布局就不能再添加到ListView中了。

View view;
ViewHolder viewHolder;
if(convertView == null){
    view = LayoutInflater.from(getContext()).inflate(resorced,parent,false);
    viewHolder = new ViewHolder();
    viewHolder.fruitImage = (ImageView)view.findViewById(...);
    view.setTag(viewHolder);
} else {
    view = convertView;
    viewHolder = (ViewHolder)view.getTag();
}
viewHolder.fruitImage.setImageResource(fruit.getImageId());
