#Kotlin数据类型

数据类型

Boolean类型

大多数情况下相当于java的基本类型boolean，只有在必要情况下才会装箱成java的装箱类型Boolean，但是这已经交给编译器来决定了，不需要我们关心。

val aBoolean:Boolean = true
val anotherBoolean:Boolean = false

Number类型

包括整形和浮点型，byte一般被当成数据流而不是整数当成二进制来看

val anInt:Int = 8
val anotherInt:Int = 0xFF
val moreInt:Int = 0b00000011
val maxInt:Int = Int.MAX_VALUE
val minInt:Int = Int.MIN_VALUE

val aLong:Long = 1417469149149817984
val anotherLong:Long = 123
val maxLong:Long = Long.MAX_VALUE
val minLong:Long = Long.MIN_VALUE

val aFloat:Float=2.0F
val anotherFloat=1E3f
val maxFloat:Float= Float.MAX_VALUE
val minFloat:Float= Float.MIN_VALUE

val aDouble:Double=3.0
val anotherDouble:Double=3.1415926
val maxDouble:Double= Double.MAX_VALUE
val minDouble:Double= Double.MIN_VALUE

val aShort:Short=127
val maxShort:Short= Short.MAX_VALUE
val minShort:Short= Short.MIN_VALUE

val maxByte:Byte= Byte.MAX_VALUE
val minByte:Byte= Byte.MIN_VALUE

fun main(args: Array<String>) {
    println(anotherInt)
    println(moreInt)
    println(maxInt)
    println(Math.pow(2.0,31.0)-1)
    println(minInt)
    println(-Math.pow(2.0,31.0))

    println(maxLong)
    println(Math.pow(2.0,63.0)-1)
    println(minLong)
    println(-Math.pow(2.0,63.0))

    println(aFloat)
    println(anotherFloat)
    println(maxFloat)
    //最小的非0正数，是正数，真正的最小值是-Float.MAX_VALUE
    println(minFloat)

    println(aDouble)
    println(anotherDouble)
    println(maxDouble)
    //最小的非0正数，是正数，真正的最小值是-Double.MAX_VALUE
    println(minDouble)

    println(aShort)
    println(maxShort)
    println(minShort)

    println(maxByte)
    println(minByte)
}

    println(string == fromChars)

    println(string === fromChars)

类与对象

什么是类

  • 类，一个抽象概念
  • 具有某些特征的事物的概括
  • 不特定指代任何一个具体的事物
  • 举例：人、车、书
  • 写法：class<类名>{<成员>}
  • 数、字符串也都是类

什么是对象

  • 是一个具体的概念，与类相对
  • 描述某一种类的具体个体
  • 举例：某些人、领导的车、Java编程思想

类与对象的关系

  • 一个类通常可以有很多个具体的对象
  • 一个对象本质上只能从属于一个类
  • 某一个人，他是工程师，但本质上还是属于人这一类

  • 对象也经常被称作“类的对象”或者“类的实例”

城市-北京、上海

国家-中国、美国

歌曲-双节棍、飞得更高

语言-汉语、日语

类的继承

  • 提取多个类的共性得到一个更抽象的类，即父类
  • 子类拥有父类的一切特征
  • 子类也可以自定义自己的特征
  • 所有类都最终继承自Any

例子：

//只有一个构造方法的时候constructor可以省略
class 妹子 constructor(性格:String,长相:String,声音:String):人(性格,长相,声音)

open class 人 (var 性格:String,var 长相:String,var 声音:String){
    init {
        println("新建一个${this.javaClass.simpleName}，性格$性格，长相$长相，声音$声音")
    }
}

fun main(args: Array<String>) {
    val myGirl:妹子= 妹子("温柔","甜美","动人")
    println(myGirl is 人)
}

空类型

String?可为空

fun getName():String?{
    return null
}

name不为空返回长度否则返回null

val name = getName()
println(name?.length)

如果name2为空则返回，否则赋值给name2

val name2:String = getName()?:return
   println(name2.length)

!!告诉编译器对字符串充分了解了不会为空

val value:String?="xxx"
println(value!!.length)

println(child)

包（package）

  • 就是命名空间
  • 包的声明必须在非注释代码的第一行

  • 类的全名：main.kotlin.市委书记.上海.市委书记

    package main.kotlin.市委书记.北京
    import main.kotlin.市委书记.上海.市委书记 as 茶水大师

    fun main(args: Array) {

    val 北京书记:市委书记=市委书记("张")
val 上海书记:茶水大师=茶水大师("李")

    }

包不一定跟路径是对应的，但一般不建议这样

区间（Range）

  • 一个数学上的概念，表示范围
  • ClosedRange的子类，IntRange最常用
  • 基本写法：

0..100 表示 [0,100]

0 until 100 表示 [0,100)

i in 0..100 判断i是否在区间[0,100]中

//闭区间[0,1024]
val range:IntRange=0..1024
//[0,1024)
val range2:IntRange=0 until 1024

val emptyRange:IntRange=0..-1

fun main(args: Array<String>) {
    println(emptyRange.isEmpty())
    println(range.contains(50))
    println(50 in range)

    for (i in range2){
        print("$i,")
    }
}

数组（Array）

跟数没有关系，就是一系列的对象

//基本数据类型避免装箱拆箱，定制了对应的Array
val arrayOfInt:IntArray= intArrayOf(1,3,5,7)
val arrayOfChar:CharArray = charArrayOf('H','e','l','l','o')
val arrayOfString:Array<String> = arrayOf("我","是","码农")
val arrayOf书记:Array<市委书记> = arrayOf(市委书记("张"),市委书记("李"))

fun main(args: Array<String>) {
    println(arrayOfInt.size)

    for (int in arrayOfInt){
        println(int)
    }

    println(arrayOf书记[1])
    arrayOf书记[1] = 市委书记("方")
    println(arrayOf书记[1])

    //第一个参数默认是,分割
    println(arrayOfChar.joinToString(""))

    //数组切片
    println(arrayOfInt.slice(1..2))
}
