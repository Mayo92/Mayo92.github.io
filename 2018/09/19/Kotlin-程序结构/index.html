Kotlin-程序结构

2018-09-19
#Kotlin程序结构

程序结构

常量与变量(val,var)

什么是常量

  • val = value，值类型
  • 类似Java的final
  • 不可能重复赋值
  • 举例：

运行时常量：val x = getX()

编译期常量：const val x = 2

什么是变量

  • var = variable
  • 举例：

var x = “Hello”//定义变量

x = “Hi”//再次赋值

类型推导

  • 编译器可以推导的类型

val string = “Hello”//推导出String类型

val int = 5//Int类型

var x = getString()+5//String类型

val与final

字节码不一样，java的final变量为编译期常量，编译器编译的时候就能知道这个变量是什么他的值是什么，只要用到这个变量的地方编译器一律对这个变量的引用换成他的值，这样代码运行效率会高一些

不加const的val定义的常量，虽然不可变但实际还是一个变量，不是编译期常量，编译器在编译期的时候不能确定它的值到底是什么，运行时还是可以用类似反射这样的手段去修改它的值

加上const则为编译期常量，这时与java的final完全一样

编译期就赋值

没有改变，值还是”HelloWorld”

去掉fianl之后需重新赋值

不加const没有直接赋值成”HelloWorld”

加了const后

函数（function）

没有返回值为Unit,可以不写

fun main(args: Array<String>) {

    println("hello ${args[0]}")
}


fun main(args: Array<String>) {

    checkArgs(args)
    val arg1 = args[0].toInt()
    val arg2 = args[1].toInt()
    println("$arg1 + $arg2 = ${sum(arg1,arg2)}")
}

fun checkArgs(args: Array<String>){
    if (args.size != 2){
        printUsage()
        //传入的值为了提示调用者程序出现问题了，根据开发者提供的对应表来查询
        System.exit(-1)//0xFF
    }

fun printUsage(){
    println("请传入两个整数参数")
}

//只返回一个表达式的函数的简洁写法
fun sum(arg1:Int,arg2:Int) = arg1 + arg2

匿名函数

val int2Long = fun(x:Int):Long{
    return x.toLong()
}

println(int2Long(3))

总结

函数也是一个类型，它与String、int一样可以赋值，可以传递。只不过它本身是一个代码块，有特殊的用处罢了

什么是函数

以特定功能组织起来的代码块

fun[函数名]([参数列表]):[返回值类型]{[函数体]}
fun[函数名]([参数列表])=[表达式]

举例

fun sayHi(name:String){print("Hi,$name")}
fun sayHi(name:String) = print("Hi,$name")

匿名函数

fun([参数列表])...

val sayHi = fun(name:String) = print("Hi,$name")

编写函数的注意事项

  • 功能要单一
  • 函数名要做到顾名思义
  • 参数个数不要太多

Lambda表达式

//Lambda表达式的返回值是表达式的最后一行的值
//(Int,Int) -> Int
val sum = {arg1:Int,arg2:Int ->
    println("$arg1 + $arg2 = ${arg1 + arg2}") //Unit
    arg1+arg2
}

//调用lambda,()相当于invoke()
println(sum(1,2))
println(sum.invoke(1,2))

简化

//函数的最后一个参数是lambda表达式 args.forEach ({ println(it) } )
//可以转换成args.forEach (){ println(it) }
//如果()内没有内容可删除 args.forEach { println(it) }
//如果传入的函数和所接收的lambda表达式类型一致 可以简化成如下
args.forEach(::println)//Array的拓展方法

终止forEach

//这里return相当于return main函数
args.forEach {
    if (it == "q") return
    println(it)
}

并不会真正终止forEach，只是把这一次的lambda表达式的调用返回了，条件前的还是继续执行

args.forEach ForEach@{
    print(it)
    if (it == "q") return@ForEach
    println(it)
}

这样写与上面截图一致

args.forEach {
    print(it)
    if (it == "q") return@forEach
    println(it)
}

真正的终止方法

这样可以真正终止，并且不会像流式数据处理循环两次

run breaking@{
    args.forEach continuing@{
        print(it)
        if (it == "q") return@breaking
        println(it)
    }
}

流式数据处理

args.takeWhile { it != "q" }.forEach(::println)

总结

什么是Lambda表达式

  • 匿名函数
  • 写法:{[参数列表] -> [函数体,最后一行是返回值]}
  • 举例：val sum = {a:Int,b:Int -> a + b}

Lambda类型表示举例

  • ()->Unit:无参，返回值为Unit
  • (Int)->Int:传入整形，返回一个整形
  • (String,(String)->String)->Boolean:传入字符串、Lambda，返回Boolean

Lambda的调用

  • 用()进行调用
  • 等价于invoke()
  • 举例：

val sum = {a:Int,b:Int -> a + b}

sum(2,3)

sum.invoke(2,3)

Lambda的简化

  • 函数参数调用时最后一个lambda可以移出去
  • 函数参数只有一个lambda，调用时小括号可以省略
  • lambda只有一个参数可默认为it
  • 入参、返回值与形参一致的函数可以用函数引用的方式作为实参传入

类成员（成员方法、成员变量）

什么是类成员

  • 属性：或者说成员变量，类范围内的变量
  • 方法：或者说成员函数，类范围内的函数

函数和方法的区别

  • 函数强调功能本身，不考虑从属
  • 方法的称呼通常是从类的角度出发
  • 叫法不同而已，不要纠结

定义方法

  • 写法与普通函数完全一致

例子

class Hello{
    fun sayHello(name:String)=print("Hello,$name")
}

定义属性

  • 构造方法参数中val/var修饰的都是属性
  • 类内部也可以定义属性

例子

class Hello(val aField:Int,notAField:Int){
    var anotherField:Float = 3f
}

notAField不是

属性访问控制

  • 属性可以定义getter/setter

例子

val a:Int = 0
    get() = field
var b:Float = 0f
    set(value){field = value}

属性初始化

  • 属性的初始化尽量在构造方法中完成
  • 无法在构造方法中初始化，尝试降级为局部变量
  • var用lateinit延迟初始化，val用lazy
  • 可空类型谨慎使用null直接初始化

例子

基本运算符

  • 任何类可以定义或者重载父类的基本运算符
  • 通过运算符对应的具名函数来定义
  • 对参数个数做要求，对参数和返回值类型不做要求
  • 不能像Scala一样定义任意运算符

例子

class Complex(var real:Double,var imaginary:Double){

    //运算符方法的重载定义要求名字对的上，前面加上operator关键字，参数个数要对的上，参数类型与返回类型不关心
    operator fun plus(other:Complex):Complex{
        return Complex(real+other.real,imaginary+other.imaginary)
    }

    operator fun plus(other:Int):Complex{
        return Complex(real+other,imaginary)
    }

    operator fun plus(other:Any):Int{
        return real.toInt()
    }

    operator fun invoke():Double{
        return Math.hypot(real,imaginary)
    }

    override fun toString(): String {
        return "$real + ${imaginary}i"
    }
}

class Book{
    //infix中缀表达式,可以不用.()这种形式调用方法

        return false

表达式（中缀表达式，分支表达式，when表达式等）

中缀表达式

  • 只有一个参数，且用infix修饰的函数

    class Book{infix fun on(place:String){….}}

    Book() on “My Desk”

分支表达式(是表达式可以有返回值不是语句)

if表达式

  • if…else

    if(a == b)…else if(a == c)…else…

  • 表达式与完备性

    val x = if(b<0) 0 else b

    val x = if(b<0) 0 //错误，赋值时，分支必须完备

when表达式

  • 加强版switch，支持任意类型
  • 支持纯表达式条件分支(类似if)
  • 表达式与完备性

例子

private const val USERNAME = "123"
private const val PASSWORD = "456"

private const val DEBUG = 1
private const val USER = 0

private const val ADMIN_NAME = "admin"
private const val ADMIN_PASSWORD = "admin"

fun main(args: Array<String>) {
    //mode初始化一次就不变了
    val mode = if(args.isNotEmpty() && args[0] == "1"){
        DEBUG
    }else{
        USER
    }

    println("请输入用户名")
    val userName = readLine()
    println("请输入用密码")
    val password = readLine()

    if(mode == DEBUG && userName == ADMIN_NAME && password == ADMIN_PASSWORD){
        println("管理员成功")
    }else if (userName == USERNAME && password == PASSWORD){
        println("成功")
    }else{
        println("失败")
    }
}


fun main(args: Array<String>) {
    val x = 5
    when(x){
        is Int -> println("hello $x")
        in 0..100 -> println("$x is in 0..100")
        !in 0..100 -> println("$x is not in 0..100")
        args[0].toInt() -> println("相等")
    }

    val mode = when{
        args.isNotEmpty() && args[0] == "1" -> 1
        else -> 0
    }
    println(mode)
}

when第一个满足就不继续往下执行了。同时判断多个条件，用,隔开

fun main(args: Array<String>) {
    for (arg in args){
        println(arg)
    }

    for ((index,value) in args.withIndex()){
        println("$index -> $value")
    }

    for (indexValue in args.withIndex()){
        println("${indexValue.index} -> ${indexValue.value}")

    val list = MyIntList()
    list.add(1)
    list.add(2)
    list.add(3)

        println(i)
    }
}

class MyIterator(val iterator:Iterator<Int>){
    operator fun next():Int{
        return iterator.next()
    }

    operator fun hasNext():Boolean{
        return iterator.hasNext()
    }

    val list = MyIntList()

}

class MyIntList{
    private val list = ArrayList<Int>()

    fun add(int:Int){
        list.add(int)
    }

    fun remove(int:Int){
        list.remove(int)
    }

    operator fun iterator():MyIterator{
        return MyIterator(list.iterator())
    }
}

异常捕获(try,catch,finally)

try…catch

  • catch分支匹配异常类型
  • 表达式，可以用来赋值

finally

  • 无论代码是否抛出异常都会执行

  • 下面的写法，先执行finally再返回

    return try{ x/y }catch(e:Exception){ 0 }finally{…}

例子

fun main(args: Array<String>) {
    try {
        val arg1 = args[0].toInt()
        val arg2 = args[1].toInt()
        println("$arg1 + $arg2 = ${sum(arg1,arg2)}")
    }catch (e:NumberFormatException){
        println("请输入整数")
    }catch (e:ArrayIndexOutOfBoundsException){
        println("请输入两个整数")
    }catch (e:Exception){
        println("出现未知异常。${e.message}")
    }finally {
        println("多谢使用")
    }
}

fun sum(arg1:Int,arg2:Int):Int{
    return arg1 + arg2
}


fun main(args: Array<String>) {
    val result = try {
        args[0].toInt() / args[1].toInt()
    }catch (e:Exception){
        e.printStackTrace()
        1
    }
    println(result)
}

具名参数，变长参数，默认参数

具名参数

  • 给函数的实参附上行参

举例:

fun sum(arg1:Int,arg2:Int)=arg1+arg2
sum(arg1=2,arg2=3)

参数顺序交换没有关系

变长参数

  • 某个参数可以接收多个值
  • 可以不作为最后一个参数
  • 如果传参时有歧义，需使用具名参数

* Spread Operator

  • 只支持展开Array
  • 只用于变长参数列表的实参
  • 不能重载
  • 不算一般意义上的运算符

默认参数

  • 为函数指定默认值
  • 可以为任意位置的参数指定默认值
  • 传参时，如果有歧义，需要使用具名参数

例子

fun main(vararg args:String) {
    //使用与数组一样
    for(arg in args){
        println(arg)
    }

//    hello(1,3,4,5,string = "Hello")
//    hello(3.0,1,3,4,5,string = "Hello")

    //*目前只支持变长参数的场景并且只能展开数组，表示把array展开变成一个一个元素传进去
    val array = intArrayOf(1,3,4,5)
    val list = arrayListOf(1,3,4,5)
//    hello(3.0,*array,string = "Hello")

//    hello(ints = *array,string = "Hello")
    hello(3.0,*array)
}

//kotlin里因为有具名参数，变长参数可以放在参数任意位置
//默认参数如果指定给比较靠前的参数，他后面的参数传入的时候只能使用具名参数的形式
fun hello(double:Double = 3.0,vararg ints:Int,string:String = "Hello"){
    println(double)
    ints.forEach(::println)
    println(string)
}
