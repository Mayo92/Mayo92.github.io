    1.安装JDK环境

    2.下载groovySDK,解压到合适位置

    打开https://groovy.apache.org/download.html，注意系统版本的要求。下载好后，主要关注bin及doc文件夹

    bin

    主要包含命令，其中groovy与java一样，groovyc与javac一样，groovysh为解释执行groovy脚本

    doc

    主要是查看html下的api及documentation

    3.配置groovy环境变量

    open ~/.zshrc

    添加

    # groovy
export PATH=${PATH}:/Users/yaoluyang/groovy-2.5.8/bin

    保存

    source ~/.zshrc

    查看是否成功

    groovy --version

    显示如下
