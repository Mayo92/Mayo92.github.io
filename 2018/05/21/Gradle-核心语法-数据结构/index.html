Gradle-核心语法-数据结构

2018-05-21
#Gradle语法

数据结构

1.groovy中列表详解

1.1 列表的定义

列表声明和初始化

//def list = new ArrayList()//java定义方式
def list = [1,2,3,4,5]
println list.class
println list.size()

定义数组的方法

列表占用了数组的定义，那么数组定义的两种方式如下

def array = [1,2,3,4,5] as int[]
int[] array2 = [1,2,3,4,5]

1.2 列表的操作

添加元素

list.add(6)
list.leftShift(7)
list << 8
println list.toString()
def plusList = list + 9
println plusList.toString()

删除元素

//[1, 2, 3, 4, 5, 6, 7, 8]
list.remove(7)//index为7
println list.toString()
list.remove((Object)7)//删除7元素
println list.toString()
//list.removeAt(7)//index为7
list.removeElement(6)//删除6元素
println list.toString()
list.removeAll{
    return it % 2 == 0//删除偶数
}
println list.toString()
println list - [1,5]//删除1、5元素

列表排序

def sortList = [6,-3,9,2,-7,1,5]
//java方法
Collections.sort(sortList)
println sortList
//定义自定义排序规则，绝对值从小到大
Comparator mc = {a,b -> a == b ? 0 : Math.abs(a) < Math.abs(b) ? -1 : 1}
Collections.sort(sortList,mc)
println sortList


//groovy方法
sortList.sort()
println sortList
sortList.sort{a,b -> a==b ? 0 : Math.abs(a) < Math.abs(b) ? 1 : -1}
println sortList


def sortStringList = ['abc','z','Hello','groovy','java','kotlin']
sortStringList.sort{it.size()}
println sortStringList


class Man{
    String name
    int age

    @Override
    String toString() {
        return name
    }
}
def wang = new Man(name: 'wang',age: 18)
def li = new Man(name: 'li',age: 36)
def zhang = new Man(name: 'zhang',age: 23)
def sortClassList = [wang,li,zhang]
sortClassList.sort{it.age}
println sortClassList.toString()

列表查找

def findList = [-3,9,6,2,-7,1,5]
//查找第一个偶数
int result = findList.find {it % 2 == 0}
println result
//查找奇数
def resultList = findList.findAll{it%2 != 0}
println resultList.toListString()
//是否有值为奇数b
def resultBoolean = findList.any{it%2 != 0}
println resultBoolean
//是否都为偶数
def resultBoolean2 = findList.every{it%2 == 0}
println resultBoolean2
//最大最小值
println findList.max()
println findList.min()
//绝对值最大最小
println findList.max{Math.abs(it)}
println findList.min{Math.abs(it)}
//统计次数，有多少个偶数
def num = findList.count {it % 2 == 0}
println num

2.groovy中映射详解（Map）

2.1 定义

key通常是字符串或者number，字符串必须是不可变字符串及单引号字符串（不指定默认为单引号）

def colors = [red:'ff0000',green:'00ff00',
            blue:'0000ff']
println colors.getClass()
//修改为HashMap或者def改成HashMap
def colors2 = [red:'ff0000',green:'00ff00',
          blue:'0000ff'] as HashMap
//不可使用这种方法，此时class为key
println colors2.class

2.2 映射的操作

获取元素值

println colors['red']
println colors.green

添加元素（没有该key时添加）

colors.yellow = 'ffff00'
println colors.toMapString()
//colors.leftShift(yellow:'iiii')
//println colors.toMapString()
//添加任意类型
colors.complex = [a:1,b:2]
println colors.toMapString()

例子

def students = [
    1:[number:'0001',name:'Bob',
        score:55,sex:'male'],
    2:[number:'0002',name:'Johnny',
       score:62,sex:'female'],
    3:[number:'0003',name:'Claire',
       score:73,sex:'female'],
    4:[number:'0004',name:'Amy',
       score:66,sex:'male']

遍历

students.each {def student ->
    println "Key is ${student.key},Value is ${student.value}"
}
//带索引的遍历（index必须为最后一个参数）
students.eachWithIndex{ def student,int index ->
    println "Index is ${index},Key is ${student.key},Value is $    {student.value}"
}
//key value遍历
students.each { value,key ->
    println "Key is ${value},Value is ${key}"
}
//index必须为最后一个参数
students.eachWithIndex { key,value,index ->
    println "Index is ${index},Key is ${key},Value is ${value}"
}

查找

//第一个符合条件的
def entry = students.find {def student ->
    return student.value.score >= 60}
println entry


def entrys = students.findAll {def student ->
    return student.value.score >= 60}
println entrys


def count = students.count {def student ->
    return student.value.score >= 60 && student.value.sex == 'male'}
println count


//输出所有及格学生的姓名
def names = students.findAll {def student ->
    return student.value.score >= 60}.
        collect {
        it.value.name
    }
println names.toListString()


//分组
def group = students.groupBy { def student ->
        return student.value.score >= 60 ? '及格' : '不及格'
}
println group.toMapString()

排序

Map的排序会产生一个新的Map，列表则是它本身的排序

def sort = students.sort{
    def student1,def student2 ->
        Number score1 = student1.value.score
        Number score2 = student2.value.score
        return score1 == score2 ? 0 : score1 > score2 ? 1 : -1
}
println sort

3.groovy中范围详解

3.1 定义

def range = 1..10
println range[0]
println range.contains(10)
println range.from
println range.to

range是list的子类，是更简单的list，操作方法与list一样

3.2 使用

遍历

range.each {
    print it
}

switch/case

def getGrade(Number number) {
    def result
    switch (number){
        case 0..<60:
            result = '不及格'
            break
        case 60..<70:
            result = '及格'
            break
        case 70..<80:
            result = '良好'
            break
        case 80..100:
            result = '优秀'
            break
    }
    return result
}

def result = getGrade(75)
println result
