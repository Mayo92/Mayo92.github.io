HTTP静态服务

安装node、npm

进入官网下载或命令下载，检查是否安装完成

node -v
npm -v

更改npm源，如不更改会导致某些资源下载失败

npm config set registry http://registry.npm.taobao.org/

安装http-server

npm install http-server -g

-g表示安装在全局，不只是当前用户能使用。如果报找不到文件的错误，使用管理员权限运行

npm install http-server -g

加载H5文件

cd到文件目录执行

//.表示在当前目录下
http-server .

如果显示结果如下，则表示本地服务器已经启动

http-server .
Starting up http-server, serving .
Available on:
http://127.0.0.1:8080
http://192.168.1.229:8080
http://192.168.2.1:8080
Hit CTRL-C to stop the server

打开浏览器，输入localhost:8080

点击文件所在文件夹即可访问H5文件

在手机端加载H5

上面结果的

  • 第一个表示本机地址，可以在电脑上直接访问
  • 第二行表示在局域网的地址，如果手机与电脑同属一个局域网，那么可以通过这个地址加载H5
  • 第三行代表电脑的WIFI地址，如果手机连接的是电脑热点，那么可以通过这个地址加载H5

关闭server

选择终端窗口，键盘输入Control+C
