HEXO总结

2019-03-06
#Blog #HEXO

1.准备环境

安装nodejs，查看node版本：
node -v
安装git，查看git版本：
git --version
安装Xcode
npm安装Hexo：
sudo npm install -g hexo(官网有坑)
初始化，在指定目录创建所需要的文件：
  • hexo init <文件夹名称>
  • cd <文件夹名称>
  • npm install
文件目录含义
  • _config.yml 配置信息
  • package.json 应用程序信息
  • scaffolds 模板文件夹
  • source 资源文件夹. _posts 目录下为你已经发表的文章. _drafts 目录下为你存放的草稿文章.
  • themes 主题文件夹

2.关联GitHub

设置SSH
新建项目，名称为用户名.github.io
输入git上传插件安装代码（安装时会提示输入github用户名及密码）
npm install hexo-deployer-git --save
在_config.yml中最后添加(在配置所有的_config.yml文件时（包括theme中的），在所有的冒号:后边都要加一个空格，否则执行hexo命令会报错.)
deploy:
      type: git
      repository:
            github: https://github.com/jikaipenggithub/jikaipenggithub.github.io
      branch: master
输入命令
hexo clean
hexo g
hexo d
打开http://你的用户名.github.io

3.安装主题

cd到blog 目录下执行如下命令：
$ git clone https://github.com/iissnan/hexo-theme-next themes/next
将blog目录下_config.yml里theme的名称landscape修改为next
cd到blog目录下执行如下命令(每次部署文章的步骤)：
hexo clean           //清除缓存文件 (db.json) 和已生成的静态文件 (public)
hexo g             //生成缓存和静态文件
hexo d             //重新部署到服务器
修改blog/_config.yml文件和blog/themes/next/_config.yml文件中对应的属性名称
cd 到 blog/themes/next/ 下执行命令 git pull 更新主题

4.新建与更新博客

bolg文件下
hexo new 'filename'
在bolg/source/_posts/下面会看到新建的博客
结构
title: "It Starts with iGaze: Visual Attention Driven Networkingwith Smart Glasses"  #博文题目
date: 2014-11-21 11:25:38      #生成时间
tags:                    #标签, 多个标签也可以使用格式[tag1, tag2, tag3,...]
- tag1
- tag2
- tag3
categories: [cat1,cat2,cat3]
---
