FFmpeg简介

音视频应用场景

  • 直播类：音视频会议、教育直播、娱乐/游戏直播等
  • 短视频：抖音、快手、小咖秀等
  • 网络视频：优酷、腾讯视频、爱奇艺等
  • 音视频通话：微信、QQ、Skype等
  • 视频监控
  • 人工智能：人脸识别、智能音箱等，更关注算法

播放器架构

FFmpeg能做什么

  • 它是一个非常优秀的多媒体框架
  • 可以运行在Linux、Mac、Windows、Android、iOS等平台上
  • 能够解码、编码、转码、复用、解复用、过滤音视频数据

FFmpeg下载编译与安装

新版本都是兼容的，老的api都可以使用。

安装完成后目录结构如下

  • libavutil：一些常用的工具
  • libavformat：对多媒体格式的解析、封装
  • libavcodec：处理音频的编解码，不做具体的编解码，只是将编解码模块插进来(h264、h265、aac等)
  • libavdevice：对音视频设备处理(音视频采集)
  • libavresample：对音频进行重采样
  • libavfilter：所有对音视频的后期处理
  • libswresample：对音频的操作
  • libswscale：对视频的操作

分页查看 ./configure –help | more

/ 的意思：目录级别分隔符。. 的意思：当前目录。./a 表示：当前目录下文件名为”a”的文件

可以更改配置属性，如下图

查看帮助的具体信息，./configure –list-encoders等，如下图

执行下面命令（配置可以根据上面更改，如调试状态可打开debug关闭优化等，–prefix为安装目录）

./configure --prefix=/Users/yaoluyang/FFmpeg --enable-gpl --enable-nonfree --enable-libfdk-aac --enable-libx264 --enable-libx265 --enable-filter=delogo --enable-debug --disable-optimizations --enable-libspeex --enable-videotoolbox --enable-shared --enable-pthreads --enable-version3 --enable-hardcoded-tables --cc=clang --host-cflags= --host-ldflags=

出现错误解决办法

nasm/yasm not found or too old. Use –disable-x86asm for a crippled build

brew install yasm

ERROR: libfdk_aac not found

brew install fdk-aac

ERROR: speex not found using pkg-config

brew install speex

ERROR: x265 not found using pkg-config

brew install x264
brew install x265

编译完成查看Makefile（按住ESC键输入:q退出）

vi Makefile

再次执行编译命令

make && make install

完成后返回之前命令安装目录，多出四个目录

  • bin目录，所有生成的程序，ffplay播放器，ffprobe多媒体侦测
  • include目录，头文件
  • lib目录，生成的动态库
  • share目录，文档相关

执行的时候

/Users/yaoluyang/FFmpeg/bin/ffmpeg

配置环境变量

open ~/.zshrc
export PATH=${PATH}:/Users/yaoluyang/FFmpeg/bin
